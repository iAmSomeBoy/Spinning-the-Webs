What is the DOM?

The DOM is a tree-like representation of an HTML document. Each HTML tag becomes a node in the tree, with parent-child relationships matching the nesting of the HTML.

What is the difference between div and section?

A <div> is a generic container with no semantic meaning. A <section> represents a themed group of content and should always have a heading inside it.
